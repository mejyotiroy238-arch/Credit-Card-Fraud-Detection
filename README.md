# Credit Card Fraud Detection

## Overview
This project uses Machine Learning to detect fraudulent credit card transactions. A Logistic Regression model is trained using the Kaggle Credit Card Fraud Detection dataset.

## Features
- Data preprocessing
- Class distribution visualization
- Logistic Regression model
- Fraud prediction
- Confusion Matrix
- Accuracy and Classification Report

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Kaggle Credit Card Fraud Detection Dataset
